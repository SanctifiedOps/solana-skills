# Marketing Skills

Narratives, announcements, content cadence, and community operations.

## Skills in This Pillar

| Skill | Description | Difficulty |
|-------|-------------|------------|
| [memecoin-narrative-engine](memecoin-narrative-engine/) | Meme frame, story arcs, symbols, cultural hooks | Intermediate |
| [launch-announcement-structures](launch-announcement-structures/) | Thread format, Telegram pinned message, FAQs | Beginner |
| [post-launch-content-cadence](post-launch-content-cadence/) | Day 0-7, week 2-4, maintaining attention ethically | Intermediate |
| [community-ops-playbook](community-ops-playbook/) | Mods, rules, expectations, handling FUD, legitimacy | Intermediate |

## When to Use These Skills

- **Creating a brand**: Start with `memecoin-narrative-engine`
- **Announcing a launch**: Use `launch-announcement-structures`
- **Maintaining momentum**: Use `post-launch-content-cadence`
- **Managing community**: Use `community-ops-playbook`

## Quick Start

**Build a meme brand:**
```
Apply memecoin-narrative-engine to create the brand kit for
a space-themed cat token. Output the origin story, 3 core
symbols, and a 2-week content calendar.
```

**Plan launch comms:**
```
Use launch-announcement-structures to draft the launch thread,
Telegram pinned message, and FAQ for a fair launch.
```

## Ethics Note

These skills focus on **legitimate marketing** for real projects:
- No fake engagement or bot manipulation
- No misleading claims or fake partnerships
- Emphasis on transparency and honest communication
- Community building over hype

See `trust/` pillar for transparency and disclosure best practices.
