# Trading Skills

Token analysis, DEX integration, whale tracking, and market dynamics for Solana traders and trenchers.

## Skills in This Pillar

| Skill | Description | Difficulty |
|-------|-------------|------------|
| [pump-fun-mechanics](pump-fun-mechanics/) | Bonding curves, graduation, migration, sniping dynamics | Intermediate |
| [jupiter-swap-integration](jupiter-swap-integration/) | Swap APIs, route optimization, slippage, frontend integration | Intermediate |
| [token-analysis-checklist](token-analysis-checklist/) | Rug detection, LP analysis, authority checks, red flags | Beginner |
| [whale-wallet-analysis](whale-wallet-analysis/) | Smart money tracking, wallet clustering, signal vs noise | Advanced |

## When to Use These Skills

- **Before buying any token**: Use `token-analysis-checklist` to verify safety
- **Building a swap UI**: Use `jupiter-swap-integration` for implementation
- **Understanding pump.fun**: Use `pump-fun-mechanics` for technical details
- **Finding alpha**: Use `whale-wallet-analysis` to track smart money

## Quick Start

**Analyze a token before buying:**
```
Use token-analysis-checklist to analyze [MINT_ADDRESS].
Check LP lock status, authority state, and holder distribution.
```

**Build a swap interface:**
```
Apply jupiter-swap-integration to create a Next.js swap component
with proper slippage handling and error states.
```
